# Muammar_PT.SGT

## System Requirements
- Node.js v18.12.0
- npm v8.19.2
- Vue 2

## Run Project
- Clone repository / salin folder project, masuk ke folder project, install dependencies, dan jalankan project:
- git clone https://github.com/username/frontend-test.git
- cd frontend-test
- npm install
- npm run serve
- Buka di browser http://localhost:8080/
